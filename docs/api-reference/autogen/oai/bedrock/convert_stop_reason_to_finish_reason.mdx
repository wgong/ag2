#### convert_stop_reason_to_finish_reason

```python
convert_stop_reason_to_finish_reason(stop_reason: str) -> Literal['stop', 'length', 'tool_calls', 'content_filter']
```

    Converts Bedrock finish reasons to our finish reasons, according to OpenAI: <br/>- stop: if the model hit a natural stop point or a provided stop sequence,
    - length: if the maximum number of tokens specified in the request was reached,
    - content_filter: if content was omitted due to a flag from our content filters,
    - tool_calls: if the model called a tool

<b>Parameters:</b>
| Name | Description |
|--|--|
| `stop_reason` | **Type:** str |
