#### calculate_correlation

```python
calculate_correlation(
    csv_path: str,
    column1: str,
    column2: str,
    method: str = 'pearson'
) -> float
```

    Calculate the correlation between two columns in a CSV file.<br/>

<b>Parameters:</b>
| Name | Description |
|--|--|
| `csv_path` | The path to the CSV file.<br/><br/>**Type:** str |
| `column1` | The name of the first column.<br/><br/>**Type:** str |
| `column2` | The name of the second column.<br/><br/>**Type:** str |
| `method` | The method used to calculate the correlation.<br/><br/>- 'pearson' (default): Pearson correlation coefficient.<br/><br/>- 'kendall': Kendall Tau correlation coefficient.<br/><br/>- 'spearman': Spearman rank correlation coefficient.<br/><br/>- callable: A custom correlation function that takes two arrays and returns a scalar.<br/><br/>**Type:** str<br/><br/>**Default:** 'pearson' |

<b>Returns:</b>
| Type | Description |
|--|--|
| float | float: The correlation coefficient between the two columns. |
