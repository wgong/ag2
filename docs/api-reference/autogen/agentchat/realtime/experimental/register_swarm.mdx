#### register_swarm

```python
register_swarm(
    *,
    realtime_agent: RealtimeAgent,
    initial_agent: ConversableAgent,
    agents: list[ConversableAgent],
    system_message: str | None = None,
    question_message: str | None = None
) -> 
```

    Create a SwarmableRealtimeAgent.<br/>

<b>Parameters:</b>
| Name | Description |
|--|--|
| `realtime_agent` | The RealtimeAgent to create the SwarmableRealtimeAgent from.<br/><br/>**Type:** [RealtimeAgent](/docs/api-reference/autogen/agentchat/realtime/experimental/RealtimeAgent) |
| `initial_agent` | The initial agent.<br/><br/>**Type:** [ConversableAgent](/docs/api-reference/autogen/ConversableAgent) |
| `agents` | The agents in the swarm.<br/><br/>**Type:** list[[ConversableAgent](/docs/api-reference/autogen/ConversableAgent)] |
| `system_message` | The system message to set for the agent.<br/><br/>If None, the default system message is used.<br/><br/>**Type:** str \| None<br/><br/>**Default:** None |
| `question_message` | The question message to set for the agent.<br/><br/>If None, the default QUESTION_MESSAGE is used.<br/><br/>**Type:** str \| None<br/><br/>**Default:** None |
