#### remove_boxed

```python
remove_boxed(string: str) -> str | None
```

    Source: https://github.com/hendrycks/math
    Extract the text within a \boxed`\{...}` environment.<br/>Example: 
    ```python
    > remove_boxed("\boxed\{\frac\{2}\{3}}")
    
    \frac\{2}\{3}
    ```

<b>Parameters:</b>
| Name | Description |
|--|--|
| `string` | **Type:** str |
