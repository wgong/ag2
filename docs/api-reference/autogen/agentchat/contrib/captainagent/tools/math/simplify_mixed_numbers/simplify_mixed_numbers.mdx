#### simplify_mixed_numbers

```python
simplify_mixed_numbers(
    numerator1,
    denominator1,
    numerator2,
    denominator2,
    whole_number1,
    whole_number2
) -> 
```

    Simplifies the sum of two mixed numbers and returns the result as a string in the format 'a b/c'.<br/>

<b>Parameters:</b>
| Name | Description |
|--|--|
| `numerator1` | The numerator of the first fraction.<br/><br/> |
| `denominator1` | The denominator of the first fraction.<br/><br/> |
| `numerator2` | The numerator of the second fraction.<br/><br/> |
| `denominator2` | The denominator of the second fraction.<br/><br/> |
| `whole_number1` | The whole number part of the first mixed number.<br/><br/> |
| `whole_number2` | The whole number part of the second mixed number.<br/><br/> |
