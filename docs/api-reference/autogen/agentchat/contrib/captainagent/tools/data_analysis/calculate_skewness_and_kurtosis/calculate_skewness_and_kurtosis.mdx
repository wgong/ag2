#### calculate_skewness_and_kurtosis

```python
calculate_skewness_and_kurtosis(csv_file: str, column_name: str) -> tuple
```

    Calculate the skewness and kurtosis of a specified column in a CSV file. The kurtosis is calculated using the Fisher definition.<br/>The two metrics are computed using scipy.stats functions.<br/>

<b>Parameters:</b>
| Name | Description |
|--|--|
| `csv_file` | The path to the CSV file.<br/><br/>**Type:** str |
| `column_name` | The name of the column to calculate skewness and kurtosis for.<br/><br/>**Type:** str |

<b>Returns:</b>
| Type | Description |
|--|--|
| tuple | tuple: (skewness, kurtosis) |
