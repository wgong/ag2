```python
MessageHistoryLimiter(max_messages: int | None = None, keep_first_message: bool = False)
```

    Limits the number of messages considered by an agent for response generation.<br/>This transform keeps only the most recent messages up to the specified maximum number of messages (max_messages).<br/>It trims the conversation history by removing older messages, retaining only the most recent messages.<br/>

<b>Parameters:</b>
| Name | Description |
|--|--|
| `max_messages` | **Type:** int \| None<br/><br/>**Default:** None |
| `keep_first_message` | **Type:** bool<br/><br/>**Default:** False |
