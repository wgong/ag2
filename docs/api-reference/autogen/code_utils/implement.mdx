#### implement

```python
implement(
    definition: str,
    configs: list[dict[str, Any]] | None = None,
    assertions: str | Callable[[str], tuple[str, float]] | None = &lt;function generate_assertions&gt;
) -> tuple[str, float]
```

    `(openai<1)` Implement a function from a definition.<br/>

<b>Parameters:</b>
| Name | Description |
|--|--|
| `definition` | The function definition, including the signature and docstr.<br/><br/>**Type:** str |
| `configs` | The list of configurations for completion.<br/><br/>**Type:** list[dict[str, typing.Any]] \| None<br/><br/>**Default:** None |
| `assertions` | The assertion code which serves as a filter of the responses, or an assertion generator.<br/><br/>**Type:** str \| Callable[[str], tuple[str, float]] \| None<br/><br/>**Default:** &lt;function generate_assertions&gt; |

<b>Returns:</b>
| Type | Description |
|--|--|
| tuple[str, float] | str: The implementation. float: The cost of the implementation. int: The index of the configuration which generates the implementation. |
